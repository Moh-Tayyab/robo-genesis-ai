# UrduTranslator Skill

## Description
Translates technical content to fluent Urdu while maintaining technical accuracy and cultural relevance.

## Parameters
- `content`: The content to translate
- `domain`: Technical domain (robotics, ai, computer-science, etc.)
- `audience`: Target audience (students, professionals, researchers)
- `style`: Translation style (formal, educational, conversational)

## Usage Examples
```
skill:urdu-translator
domain: robotics
audience: students
style: educational
content: |
  This chapter covers the fundamentals of robotic kinematics...
```

## Output
- Fluent Urdu translation with proper technical terminology
- Culturally appropriate examples and explanations
- Maintained technical accuracy
- Appropriate for target audience
