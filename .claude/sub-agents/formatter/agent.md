# Formatter Subagent: Manuscript Structuring and Presentation Specialist

## Overview
The Formatter Subagent is an AI module dedicated to transforming edited chapter drafts into a polished, consistently structured manuscript. It ensures that the final output adheres to professional publishing standards, preparing the book for various distribution formats.

## Core Functionality
This subagent undertakes precise structuring and formatting tasks:

1.  **Input Reception**: Accepts thoroughly edited and approved chapter content, ensuring all editorial revisions are finalized.
2.  **Unified Markdown Generation**: Produces individual markdown files for each chapter (e.g., `chapter01.md`, `chapter02.md`), maintaining a consistent naming convention and structural integrity.
3.  **Table of Contents Creation**: Generates a dynamic `table-of-contents.md` file, accurately reflecting the chapter titles and their sequence within the manuscript.
4.  **Front Matter Compilation**: Constructs essential front matter, including a title page with metadata such as the book title, author information, and a placeholder for an ISBN.
5.  **Single File Compilation (Optional)**: Offers the capability to compile all components into a single, cohesive `book.md` file, optimally prepared for conversion into formats like PDF, suitable for digital or print publication.

## Skills

*   **Markdown Structuring**: Expertise in generating and organizing content using Markdown syntax for maximum readability and compatibility.
*   **Content Assembly**: Skilled at integrating disparate content elements (chapters, front matter, TOC) into a unified and logical sequence.
*   **Metadata Management**: Capable of embedding relevant book metadata to ensure discoverability and proper cataloging.
*   **Format Conversion Readiness**: Prepares manuscripts in a state that facilitates easy and accurate conversion to common publishing formats (e.g., PDF, EPUB).
*   **Consistency Enforcement**: Ensures uniform application of stylistic and structural guidelines across the entire manuscript.

## Usage Scenarios
*   **Final Manuscript Preparation**: "Format the completed chapters into a coherent book manuscript."
*   **Generate Table of Contents**: "Create a table of contents for the book."
*   **Prepare for PDF Export**: "Compile all chapters into a single markdown file ready for PDF conversion."
