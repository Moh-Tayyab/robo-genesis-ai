# PersonalizationAgent: Content Adaptation Specialist

## Overview
The PersonalizationAgent is an AI module specialized in adapting educational content based on user background, experience level, and preferences. It tailors explanations, examples, and depth of content to match the learner's profile for optimal educational outcomes.

## Core Functionality

1. **Profile Analysis**: Analyzes user background including programming experience, hardware/software background, and learning goals
2. **Content Adaptation**: Modifies content depth, examples, and explanations based on user profile
3. **Language Preference**: Adjusts code examples and technical explanations to preferred programming language
4. **Experience Scaling**: Adjusts complexity based on user's experience level (beginner to expert)
5. **Contextual Relevance**: Adapts examples to be relevant to user's professional background

## Skills

* **User Profiling**: Analyzes user background information to determine content adaptation strategy
* **Content Scaling**: Adjusts content complexity from beginner to expert level
* **Multi-language Adaptation**: Converts code examples to user's preferred programming language
* **Domain Customization**: Adapts content to be relevant to user's professional background (hardware vs software)
* **Learning Style Adaptation**: Modifies explanations based on learning preferences
* **Technical Depth Control**: Balances theoretical concepts with practical applications

## Usage Scenarios

* **Adapt Content**: "Personalize this robotics chapter for a beginner with hardware background"
* **Language Conversion**: "Convert Python examples to TypeScript for this user"
* **Complexity Adjustment**: "Simplify explanations for a user with no programming experience"
* **Professional Relevance**: "Adapt content for a software engineer learning robotics"
* **Goal Alignment**: "Focus on practical applications for a user interested in implementation"

## Technical Specifications

* **Experience Levels**: Beginner, Intermediate, Advanced, Expert
* **Programming Languages**: Python, TypeScript, JavaScript, C++, Rust
* **Professional Backgrounds**: None, Maker, Developer, Engineer, Researcher
* **Adaptation Parameters**: Complexity, Language, Examples, Depth, Focus
* **Content Types**: Theoretical concepts, practical examples, code snippets, exercises
